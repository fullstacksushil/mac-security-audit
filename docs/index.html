mac-security-audit

Security audit for macOS machines running local AI.
One command. Zero dependencies. Works on any Mac.

Install

Audit only:

curl -fsSL https://raw.githubusercontent.com/fullstacksushil/mac-security-audit/main/mac-security-audit.sh | bash

Full audit + auto-fix (recommended):

curl -fsSL https://raw.githubusercontent.com/fullstacksushil/mac-security-audit/main/mac-security-audit.sh | sudo bash -s -- --fix

Why

Mac Minis are the new home server for local AI. Most setups have problems:

  • Ollama bound to 0.0.0.0 — your models are on the network
  • AI agent gateways with weak auth tokens
  • Docker containers exposed on all interfaces
  • Tailscale Funnel publishing services to the internet
  • Orphaned agent configs with API keys on disk
  • Firewall off, FileVault off, Remote Desktop on
